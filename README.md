# Double Side Company Management System

## 📌 Overview

The **Double Side Company Management System** is a Java-based desktop application designed to manage and organize company operations efficiently with main 2 interfaces(For Company AND for Custommers).
It provides tools for handling employee data, company activities, manage salary, create own quotation and basic administrative tasks in a structured and user-friendly way.

---

## 🎯 Objectives

* Simplify company management processes
* Provide a centralized system for handling records
* Demonstrate Object-Oriented Programming (OOP) concepts in Java

---

## 🚀 Features

* 👨‍💼 Employee Management (Add, Update, Delete, View)
* 📋 Record Management
* 🔍 Search and filtering functionality
* 🖥️ User-friendly interface (Java Swing / GUI)
* 📊 Salary calcutation & Basic system organization for company operations
*    Create own quotation(Customer side interface)

---

## 🛠️ Technologies Used

* **Java** (Core Java, OOP Concepts)
* **SQL**
* **Java Swing** (for GUI)
* **NetBeans / IntelliJ IDEA** (recommended IDEs)

---

## 🧱 Project Structure

```
src/mainpackage/
 ├── SlaryCalc/ # Salary calculator
 ├── Bill/      # quotation CREATOR
 ├── userinterface/ # User interface (Swing forms)
 └── login/         # Main class to run the application
```
